## Overview
Frontend for the Todo Management System built with React, TypeScript, and Vite.

- Add, edit, toggle, and delete todos
- React Router navigation
- Context API state management
- Bonus Challenge A: max 5 active tasks, FIFO completion, 15-second archive
- Bonus Challenge B: blockchain verification with tamper detection
